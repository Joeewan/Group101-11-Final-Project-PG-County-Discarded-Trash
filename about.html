ABOUT WEBSITE

This website displays a map visualization of data from LitterTRAK dataset that contains information about yearly data on litter and trash discarded from different regions of Prince George’s County from Prince George’s County OpenData web repository.

HOW TO NAVIGATE

On the homepage, use the search bar to type a specific zipcode within the scope of PG County to see markers indicating the trash data within the zipcode's region.
